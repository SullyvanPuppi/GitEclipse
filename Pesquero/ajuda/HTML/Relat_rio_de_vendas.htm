Relatório de vendas

 

Para gerar relatórios de vendas o sistema permite de quatro formas.

 

  • Por consulta

Clicando em relatório o sistema gerará o relatório referente a consulta utilizada no formulário de consulta.

 

  • Por especificação

No menu Vendas/Relatórios o sistema exibirá uma caixa de diálogo dando opção de gerar relatório detalhado onde se tem todos os dados vendas ou relatório resumido onde será emitido com dados básicos de todas as vendas realizadas e também extrato onde se abrirá uma nova janela de diálogo solicitando o nome do funcionário e o período de vendas para gerar um extrato de vendas deste funcionário.

 

  • Individual

Permite gerar relatório referente a venda em exibição no formulário.

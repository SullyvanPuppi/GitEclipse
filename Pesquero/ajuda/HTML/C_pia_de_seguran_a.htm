Cópia de segurança

 

O sistema permite realizar cópia de segurança da base de dados através do menu Sistema/Cópia de segurança;

Primeiro abre-se uma janela perguntando o que deseja fazer se gerar cópia de segurança dos dados ou recuperar a base de dados.

 

 

  • Cópia de segurança

É necessário selecionar o local onde se deseja salvar a cópia de segurança e clicar em salvar.

 

  • Recuperando cópia de segurança

É necessário selecionar o local onde esta a cópia de segurança e clicar em salvar.
